Projects

CPU Track

Built a tool to track CPU usage of processes using C++ and WinAPI. I wrote a program to track CPU usage of multiple processes using C++ and WinAPI library. I Collected data on the CPU usage of different processes running, using the Performance Counter API. I Used multi-threading to store the metrics on a CSV while also updating the GUI that presented the data. This project provided useful info on which processes have high spikes in usage, and which processes utilized the CPU the most.

Aggie Professor Search

Constructed a full-stack web application for Texas A&M Students to better gauge classes they want to take, by allowing them to view Professors' GPA distribution, and rating, per ratemyprofessors.com. Implemented the Python libraries Selenium and Scrapy to scrape the Texas A&M Grade distributions website and Ratemyprofessors.com. Utilized React to create the front-end portion of the website. Solved the problem of having to visit multiple websites and having multiple pages open to view data on the quality and difficulty of different classes at Texas A&M University.

Tally

I created a social media app, Tally, with React Native, and am deploying the expo platform. I implemented the user interface including the login screen, different pages, and navigation bar. Collaborated with a team with no prior knowledge of react native, and had a working product in a few months. The goal of the app was to increase collaboration between teams, or even just increase comradery in roommates by “gamifying” common tasks that needed to be done. This was done by allowing users to create or join different groups, as well as add tasks for each group. Each task that was completed rewarded the user with points.

Go Phish

Created a custom inbox, using Tkinter GUI, and the Gmail API to access the user’s emails. The Custom inbox forwards all the emails from the user’s Gmail account, and then cross-references all the links in a given email with known malware links, and uses an AI model we created to detect how likely the email is to be a phishing attempt.
